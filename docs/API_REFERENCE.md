# API Reference: Phase 5 Production Features\n\nQuick reference for all new endpoints added in Phase 5.\n\n## Authentication\n\nAll endpoints require `Authorization: Bearer {token}` header unless marked as [Public].\n\n## Loyalty System\n\n### GET `/api/loyalty/profile`\nGet current user's loyalty profile and tier\n\n**Response:**\n```json\n{\n  \"userId\": \"uuid\",\n  \"totalPoints\": 5000,\n  \"currentTier\": \"gold\",\n  \"tierProgress\": 75,\n  \"pointsExpiring\": {\n    \"within30Days\": 100,\n    \"within90Days\": 250\n  },\n  \"tierBenefits\": {\n    \"discount\": \"15%\",\n    \"freeShipping\": true,\n    \"prioritySupport\": true\n  }\n}\n```\n\n### GET `/api/loyalty/benefits`\nView tier benefits and eligibility\n\n**Response:**\n```json\n{\n  \"bronze\": { \"discount\": \"5%\", \"freeShipping\": false },\n  \"silver\": { \"discount\": \"10%\", \"freeShipping\": false },\n  \"gold\": { \"discount\": \"15%\", \"freeShipping\": true },\n  \"platinum\": { \"discount\": \"20%\", \"freeShipping\": true, \"concierge\": true }\n}\n```\n\n### POST `/api/loyalty/redeem`\nRedeem loyalty points for discount\n\n**Body:**\n```json\n{\n  \"points\": 500,\n  \"orderId\": \"uuid\"\n}\n```\n\n**Response:**\n```json\n{\n  \"success\": true,\n  \"pointsRedeemed\": 500,\n  \"discountAmount\": 5.00,\n  \"remainingPoints\": 4500\n}\n```\n\n---\n\n## Recommendations\n\n### GET `/api/recommendations?limit=12`\nGet personalized product recommendations\n\n**Query Parameters:**\n- `limit` (number, default: 12): Max results to return\n- `category` (string, optional): Filter by category\n\n**Response:**\n```json\n{\n  \"recommendations\": [\n    {\n      \"productId\": \"uuid\",\n      \"name\": \"Wireless Headphones\",\n      \"score\": 0.94,\n      \"reason\": \"based_on_browsing_history\"\n    }\n  ],\n  \"totalCount\": 45,\n  \"generatedAt\": \"2024-01-15T10:30:00Z\"\n}\n```\n\n### GET `/api/recommendations/trending`\nGet trending products\n\n**Query Parameters:**\n- `timeframe` (enum: \"24h\", \"7d\", \"30d\", default: \"7d\")\n- `category` (string, optional)\n- `limit` (number, default: 20)\n\n**Response:**\n```json\n{\n  \"trending\": [\n    {\n      \"productId\": \"uuid\",\n      \"name\": \"Item Name\",\n      \"trendingScore\": 0.88,\n      \"viewsLast7Days\": 5000,\n      \"purchasesLast7Days\": 450\n    }\n  ]\n}\n```\n\n### POST `/api/recommendations/:productId/interact`\nTrack user interaction (view, cart, purchase)\n\n**Body:**\n```json\n{\n  \"eventType\": \"viewed\",\n  \"duration\":u 30\n}\n```\n\n**Event Types:** `viewed`, `added_to_cart`, `purchased`, `returned`\n\n---\n\n## Returns & RMA\n\n### POST `/api/returns`\nInitiate a return request\n\n**Body:**\n```json\n{\n  \"orderId\": \"uuid\",\n  \"orderItemId\": \"uuid\",\n  \"quantity\": 1,\n  \"reason\": \"defective\",\n  \"description\": \"Item arrived with broken screen\"\n}\n```\n\n**Response:**\n```json\n{\n  \"returnId\": \"uuid\",\n  \"status\": \"requested\",\n  \"estimatedRefund\": 49.99,\n  \"returnAddress\": \"123 Warehouse St, City, State\"\n}\n```\n\n### GET `/api/returns/my-returns`\nGet user's return history\n\n**Response:**\n```json\n{\n  \"returns\": [\n    {\n      \"returnId\": \"uuid\",\n      \"orderId\": \"uuid\",\n      \"product\": { \"id\": \"uuid\", \"name\": \"..." },\n      \"status\": \"refunded\",\n      \"refundAmount\": 49.99,\n      \"refundDate\": \"2024-01-15T10:30:00Z\"\n    }\n  ],\n  \"total\": 3\n}\n```\n\n### GET `/api/returns/order/:orderId`\nGet returns for specific order\n\n### PUT `/api/returns/:returnId/approve` [Admin]\nApprove a return request\n\n**Body:**\n```json\n{\n  \"notes\": \"Return approved, shipping label sent\"\n}\n```\n\n### PUT `/api/returns/:returnId/refund` [Admin]\nProcess refund for approved return\n\n**Body:**\n```json\n{\n  \"returnedCondition\": \"good\",\n  \"notes\": \"Item verified, initiating refund\"\n}\n```\n\n---\n\n## Fraud Detection\n\n### POST `/api/fraud/:fraudId/review` [Admin]\nReview and approve/reject flagged order\n\n**Body:**\n```json\n{\n  \"decision\": \"approved\",\n  \"notes\": \"Manual review: customer verified via phone\",\n  \"falsPositive\": false\n}\n```\n\n**Decisions:** `approved`, `rejected`\n\n---\n\n## Dynamic Pricing\n\n### GET `/api/dynamic-pricing/price?productId=uuid&quantity=5`\nGet dynamic price based on user tier and quantity\n\n**Query Parameters:**\n- `productId` (uuid, required)\n- `quantity` (number, optional, default: 1)\n\n**Response:**\n```json\n{\n  \"productId\": \"uuid\",\n  \"basePrice\": 99.99,\n  \"loyaltyDiscount\": 10.00,\n  \"volumeDiscount\": 5.00,\n  \"demandMultiplier\": 1.1,\n  \"finalPrice\": 84.99,\n  \"savings\": 15.00,\n  \"savingsPercent\": 15.0\n}\n```\n\n### POST `/api/dynamic-pricing/adjust` [Admin]\nAdjust pricing rules\n\n**Body:**\n```json\n{\n  \"productId\": \"uuid\",\n  \"rules\": {\n    \"loyaltyTiers\": [\n      { \"tier\": \"bronze\", \"discount\": 0.05 },\n      { \"tier\": \"gold\", \"discount\": 0.15 }\n    ],\n    \"volumeBreaks\": [\n      { \"quantity\": 5, \"discount\": 0.05 },\n      { \"quantity\": 20, \"discount\": 0.10 }\n    ]\n  }\n}\n```\n\n---\n\n## Seller Commission\n\n### GET `/api/sellers/commissions/my-commissions`\nView seller's earning commissions\n\n**Response:**\n```json\n{\n  \"commissions\": [\n    {\n      \"orderId\": \"uuid\",\n      \"productPrice\": 100.00,\n      \"commissionRate\": 15,\n      \"commissionAmount\": 15.00,\n      \"taxAmount\": 2.70,\n      \"netCommission\": 12.30,\n      \"status\": \"pending\",\n      \"createdAt\": \"2024-01-15T10:30:00Z\"\n    }\n  ],\n  \"totalPending\": 500.00,\n  \"totalApproved\": 2500.00,\n  \"totalPaid\": 10000.00\n}\n```\n\n### GET `/api/sellers/commissions/my-payouts`\nView seller's payout history\n\n**Response:**\n```json\n{\n  \"payouts\": [\n    {\n      \"payoutId\": \"uuid\",\n      \"period\": \"2024-01\",\n      \"totalCommissions\": 5000.00,\n      \"netPayout\": 4100.00,\n      \"status\": \"completed\",\n      \"paymentDate\": \"2024-02-01T10:30:00Z\",\n      \"paymentMethod\": \"bank_transfer\"\n    }\n  ]\n}\n```\n\n### POST `/api/sellers/commissions/calculate/:orderId` [Admin]\nManually calculate commission for order\n\n### POST `/api/sellers/commissions/process-payouts` [Admin]\nTrigger monthly payout processing\n\n**Body:**\n```json\n{\n  \"sellerId\": \"uuid (optional, if null process all sellers)\" \n}\n```\n\n### GET `/api/sellers/commissions/settlements` [Admin]\nView all settlement records\n\n**Query Parameters:**\n- `sellerId` (uuid, optional)\n- `month` (string, format: \"2024-01\", optional)\n\n---\n\n## Notifications\n\n### GET `/api/notifications/preferences`\nGet notification preferences\n\n**Response:**\n```json\n{\n  \"email\": {\n    \"enabled\": true,\n    \"orderUpdates\": true,\n    \"marketing\": false,\n    \"weekly_digest\": true\n  },\n  \"sms\": {\n    \"enabled\": true,\n    \"orderUpdates\": true,\n    \"promotions\": false\n  },\n  \"push\": {\n    \"enabled\": false,\n    \"orderUpdates\": true,\n    \"recommendations\": false\n  },\n  \"inApp\": {\n    \"enabled\": true,\n    \"all\": true\n  },\n  \"dnd\": {\n    \"enabled\": true,\n    \"startHour\": 22,\n    \"endHour\": 8,\n    \"timezone\": \"America/New_York\"\n  }\n}\n```\n\n### PUT `/api/notifications/preferences`\nUpdate notification preferences\n\n**Body:**\n```json\n{\n  \"email\": { \"marketing\": true },\n  \"sms\": { \"enabled\": false },\n  \"dnd\": {\n    \"enabled\": true,\n    \"startHour\": 22,\n    \"endHour\": 8,\n    \"timezone\": \"UTC\"\n  }\n}\n```\n\n---\n\n## Response Status Codes\n\n- `200`: Success\n- `201`: Created\n- `400`: Bad Request (validation error)\n- `401`: Unauthorized (missing/invalid token)\n- `403`: Forbidden (insufficient permissions)\n- `404`: Not Found\n- `409`: Conflict (duplicate/constraint violation)\n- `429`: Too Many Requests (rate limited)\n- `500`: Server Error\n- `503`: Service Unavailable\n\n## Common Error Response\n\n```json\n{\n  \"error\": \"Error message\",\n  \"code\": \"ERROR_CODE\",\n  \"statusCode\": 400,\n  \"details\": {\n    \"field\": \"validation error details\"\n  }\n}\n```\n\n## Rate Limits\n\n- Global: 300 requests per minute per IP\n- Auth endpoints: 10 requests per minute\n- Payment endpoints: 5 requests per minute\n\n## Pagination\n\nEndpoints that return lists support:\n\n```\nGET /api/endpoint?page=1&limit=20&sort=createdAt:desc\n```\n\n**Response:**\n```json\n{\n  \"items\": [...],\n  \"pagination\": {\n    \"page\": 1,\n    \"limit\": 20,\n    \"total\": 250,\n    \"pages\": 13\n  }\n}\n```\n\n## Webhook Events (for real-time integrations)\n\n- `loyalty.points_awarded`\n- `fraud.order_flagged`\n- `fraud.order_approved`\n- `return.requested`\n- `return.approved`\n- `return.refunded`\n- `commission.calculated`\n- `payout.processed`\n"
