# Integration Guide: Phase 5 Production Features\n\nThis guide covers how to integrate all Phase 5 production-level features into your application workflow.\n\n## 1. Database Migrations\n\n### Running SQL Migrations\n\n```bash\n# Connect to PostgreSQL and run migrations\npsql -U postgres -d ecommerce < sql/migrations/001-create-seller-commission-tables.sql\npsql -U postgres -d ecommerce < sql/migrations/002-create-advanced-features-tables.sql\n```\n\n### Using Knex Migrations (Alternative)\n\n```bash\nnpm install --legacy-peer-deps\n# Create migration files\nknex migrate:make create_seller_commissions --knexfile src/config/knexfile.js\n# Run migrations\nknex migrate:latest --knexfile src/config/knexfile.js\n```\n\n## 2. Route Registration\n\nAll new routes are automatically registered in `src/api/register-routes.js`. The endpoints are:\n\n```\nGET    /api/loyalty/profile                  - Get loyalty profile\nGET    /api/loyalty/benefits                 - View tier benefits\nPOST   /api/loyalty/redeem                   - Redeem points\n\nGET    /api/recommendations                  - Personalized recommendations\nGET    /api/recommendations/trending         - Trending products\nPOST   /api/recommendations/:productId/interact - Track user interaction\n\nPOST   /api/returns                          - Create return request\nGET    /api/returns/my-returns               - View user returns\nGET    /api/returns/order/:orderId           - Get order returns\nPUT    /api/returns/:returnId/approve        - [Admin] Approve return\nPUT    /api/returns/:returnId/refund         - [Admin] Process refund\n\nPOST   /api/fraud/:fraudId/review            - [Admin] Review flagged order\n\nGET    /api/dynamic-pricing/price            - Get dynamic price\nPOST   /api/dynamic-pricing/adjust           - [Admin] Adjust pricing rules\n\nGET    /api/sellers/commissions/my-commissions  - View seller commissions\nGET    /api/sellers/commissions/my-payouts      - View payout history\nPOST   /api/sellers/commissions/calculate/:orderId - [Admin] Calculate commission\nPOST   /api/sellers/commissions/process-payouts   - [Admin] Process payouts\nGET    /api/sellers/commissions/settlements      - [Admin] View settlements\n\nGET    /api/notifications/preferences         - Get notification preferences\nPUT    /api/notifications/preferences         - Update preferences\n```\n\n## 3. Loyalty System Integration\n\n### Hook into Order Payment Success\n\nIn `src/modules/order/services/order.service.js`:\n\n```javascript\nconst { LoyaltyService } = require('../loyalty/services/loyalty.service');\n\n// After payment is confirmed\nconst orderTotal = await calculateOrderTotal(order);\nconst pointsEarned = Math.floor(orderTotal); // 1 point per $1\nawait LoyaltyService.awardPoints(order.user_id, pointsEarned, {\n  orderId: order.id,\n  transactionType: 'purchase',\n  description: `Purchase Order #${order.id}`\n});\n```\n\n## 4. Recommendation Engine Integration\n\n### Track User Interactions\n\n```javascript\nconst { RecommendationService } = require('../recommendation/services/recommendation.service');\n\n// On product view\nawait RecommendationService.trackInteraction(userId, productId, 'viewed');\n\n// On add to cart\nawait RecommendationService.trackInteraction(userId, productId, 'added_to_cart');\n\n// On purchase\nawait RecommendationService.trackInteraction(userId, productId, 'purchased');\n```\n\n### Get Recommendations\n\n```javascript\nconst recommendations = await RecommendationService.getRecommendations(userId, limit=12);\n```\n\n## 5. Fraud Detection Integration\n\n### Validate Order Before Payment\n\nIn `src/modules/order/services/order.service.js`:\n\n```javascript\nconst { FraudDetectionService } = require('../fraud/services/fraud-detection.service');\n\n// Before processing payment\nconst fraudAnalysis = await FraudDetectionService.analyzeOrder(order, {\n  paymentMethod: order.payment_method,\n  cardCountry: cardMetadata.country,\n  userIp: req.ip,\n  deviceFingerprint: req.headers['x-device-fingerprint']\n});\n\nif (fraudAnalysis.riskLevel === 'critical') {\n  // Flag for review or reject\n  return res.status(403).json({ error: 'Order flagged for review' });\n}\n```\n\n## 6. Dynamic Pricing Integration\n\n### Get Dynamic Price When Retrieving Products\n\nIn product controller:\n\n```javascript\nconst { DynamicPricingService } = require('../pricing/services/dynamic-pricing.service');\nconst { LoyaltyService } = require('../loyalty/services/loyalty.service');\n\n// Get product\nconst product = await Product.findById(productId);\n\n// Get user loyalty tier if authenticated\nlet loyaltyTier = 'standard';\nif (req.auth) {\n  const loyaltyProfile = await LoyaltyService.getProfile(req.auth.sub);\n  loyaltyTier = loyaltyProfile.currentTier;\n}\n\n// Calculate dynamic price\nconst dynamicPrice = await DynamicPricingService.calculatePrice(\n  product.basePrice,\n  productId,\n  req.auth?.sub,\n  loyaltyTier\n);\n\nproduct.displayPrice = dynamicPrice.finalPrice;\nproduct.originalPrice = product.basePrice;\nproduct.discount = dynamicPrice.discountAmount;\n```\n\n## 7. Seller Commission Integration\n\n### Calculate Commission After Order Fulfillment\n\n```javascript\nconst { CommissionService } = require('../seller/services/commission.service');\n\n// After order item is fulfilled\nconst commission = await CommissionService.calculateCommission(orderId);\n\n// Create payout batch (run as scheduled task in cron job)\nconst payouts = await CommissionService.processBatchPayouts();\n```\n\n### Add to Cron Job\n\nIn `src/infrastructure/cron/register-cron.js`:\n\n```javascript\nconst { CommissionService } = require('../../modules/seller/services/commission.service');\n\n// Every 1st of the month, process seller payouts\nconst commissionJob = cron.schedule('0 0 1 * *', async () => {\n  try {\n    await CommissionService.processBatchPayouts();\n    logger.info('Monthly seller payouts processed');\n  } catch (error) {\n    logger.error('Error processing payouts:', error);\n  }\n});\n```\n\n## 8. Return/RMA Integration\n\n### Auto-Refund on Return Approval\n\nIn return service:\n\n```javascript\nconst { ReturnService } = require('../returns/services/return.service');\n\n// When return is received and verified\nawait ReturnService.approveReturn(returnId, {\n  receivedCondition: 'good',\n  notes: 'Item verified, initiating refund'\n});\n\n// This will:\n// 1. Update return status to \"refunded\"\n// 2. Process refund to original payment method\n// 3. Create ledger entry\n// 4. Notify customer\n```\n\n## 9. Cache Integration\n\n### Cache Hot Data\n\n```javascript\nconst { RedisClient } = require('../infrastructure/cache/redis-client');\n\n// Cache product data for 1 hour\nconst cacheKey = RedisClient.buildCacheKey('product', productId);\nlet product = await RedisClient.get(cacheKey);\n\nif (!product) {\n  product = await Product.findById(productId);\n  await RedisClient.set(cacheKey, product, 3600); // 1 hour TTL\n}\n\n// Cache user profile\nconst userKey = RedisClient.buildCacheKey('user', userId);\nawait RedisClient.set(userKey, userProfile, 1800);\n\n// Cache cart\nconst cartKey = RedisClient.buildCacheKey('cart', userId);\nawait RedisClient.set(cartKey, cart, 3600);\n\n// Invalidate on update\nawait RedisClient.delete(cacheKey);\n```\n\n## 10.
